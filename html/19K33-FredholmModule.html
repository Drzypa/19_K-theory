Fredholm module


Fredholm modules represent abstract elliptic pseudo-differential operators.

Definition 1.

An odd Fredholm module (ℋ,F) over a C*-algebra A is given by an involutive representation π of A on a Hilbert space ℋ, together with an operator F on ℋ such that F=F*, F2=1⁢I and [F,π⁢(a)]∈𝕂⁢(ℋ) for all a∈A.

Definition 2.

An even Fredholm module (ℋ,F,Γ) is given by an odd Fredholm module (ℋ,F) together with a ℤ2-grading Γ on ℋ, Γ=Γ*, Γ2=1⁢I, such that Γ⁢π⁢(a)=π⁢(a)⁢Γ and Γ⁢F=-F⁢Γ.

Definition 3.

A Fredholm module is called degenerate if [F,π⁢(a)]=0 for all a∈A. Degenerate Fredholm modules are homotopic to the 0-module.

Example 1 (Fredholm modules over C)

An even Fredholm module (H,F,Γ) over C is given by

ℋ = ℂk⊕ℂk with ⁢π⁢(a)=(a⁢1⁢Ik000),
F = (01⁢Ik1⁢Ik0),
Γ = (1⁢Ik00-1⁢Ik).
